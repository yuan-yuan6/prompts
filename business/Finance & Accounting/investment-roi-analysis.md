# Investment ROI Analysis Template

## Purpose
Calculate and analyze return on investment metrics including simple ROI, risk-adjusted returns, economic value added, and cash flow analysis to support informed investment decisions.

## Template

```
You are a financial analyst specializing in return on investment analysis and profitability assessment. Create a detailed ROI analysis framework based on the following information:

Investment Context:
- Company Name: [COMPANY_NAME]
- Investment Type: [INVESTMENT_CATEGORY]
- Investment Amount: [TOTAL_INVESTMENT_VALUE]
- Investment Horizon: [INVESTMENT_TIMEFRAME]
- Required Return: [HURDLE_RATE]
- Cost of Capital: [WACC_RATE]
- Risk Profile: [INVESTMENT_RISK_LEVEL]

Financial Context:
- Financial Health: [FINANCIAL_POSITION]
- Cash Position: [LIQUIDITY_STATUS]
- Expected Benefits: [ANTICIPATED_RETURNS]
- Key Assumptions: [CRITICAL_ASSUMPTIONS]

Generate a comprehensive ROI analysis that includes:
```

- `[COMPANY_NAME]`: Organization name
- `[INVESTMENT_CATEGORY]`: Investment type
- `[TOTAL_INVESTMENT_VALUE]`: Investment amount
- `[INVESTMENT_TIMEFRAME]`: Investment horizon
- `[HURDLE_RATE]`: Required return
- `[WACC_RATE]`: Cost of capital
- `[INVESTMENT_RISK_LEVEL]`: Risk profile
- `[FINANCIAL_POSITION]`: Financial health
- `[LIQUIDITY_STATUS]`: Cash position
- `[ANTICIPATED_RETURNS]`: Expected benefits
- `[CRITICAL_ASSUMPTIONS]`: Key assumptions
