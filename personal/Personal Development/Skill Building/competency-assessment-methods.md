# Competency Assessment Methods

## Purpose
Implement comprehensive competency assessment methods including multi-rater feedback, skill assessments, behavioral interviews, and performance evaluation.

## Template

```
You are a competency assessment specialist. Design assessment program for:

Assessment Context:
- Target Population: [TARGET_GROUP]
- Competencies to Assess: [COMPETENCY_LIST]
- Assessment Purpose: [PURPOSE]
- Timeline: [TIMELINE]

Generate comprehensive assessment approach:

## ASSESSMENT METHODS

### Self-Assessment
- Reflection questionnaires
- Skill rating scales
- Goal setting exercises
- Strength/weakness analysis
- Development planning

### Manager Assessment
- Performance reviews
- Competency ratings
- Behavioral observations
- Goal achievement evaluation
- Potential assessment

### 360-Degree Feedback
- Peer feedback
- Direct report feedback
- Cross-functional input
- Customer/stakeholder feedback
- Comprehensive view

### Skills Testing
- Technical skill tests
- Simulations and scenarios
- Work samples and portfolios
- Certification exams
- Practical demonstrations

### Behavioral Interviews
- STAR method (Situation, Task, Action, Result)
- Competency-based questions
- Behavioral examples
- Critical incident technique
- Structured interview guides

### Performance Data
- Objective metrics and KPIs
- Project outcomes
- Quality measures
- Productivity indicators
- Achievement records

## ASSESSMENT PROCESS

### Planning
- Define assessment objectives
- Select appropriate methods
- Develop assessment tools
- Train assessors
- Communicate to participants

### Execution
- Administer assessments
- Collect data systematically
- Ensure fairness and consistency
- Maintain confidentiality
- Document results

### Analysis
- Aggregate results
- Identify patterns and trends
- Compare to benchmarks
- Generate insights
- Prepare reports

## DELIVERABLES

- Assessment methodology document
- Assessment tools and instruments
- Assessor training materials
- Data collection templates
- Analysis and reporting framework
- Feedback templates

Ensure assessments are valid, reliable, fair, and actionable.
```

## Usage Example
Use for talent assessment, performance evaluation, or development planning.
