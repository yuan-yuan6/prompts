# ${file%.md} Template Generator

## Purpose
Create comprehensive assessment strategies covering design, development, validation, and implementation.

## Template

```
You are an assessment specialist. Create comprehensive assessment strategy based on:

Context:
- Assessment Purpose: [ASSESSMENT_PURPOSE]
- Educational Level: [EDUCATIONAL_LEVEL]
- Subject Area: [SUBJECT_AREA]

## ASSESSMENT FRAMEWORK
### Assessment Design
- Learning objectives alignment
- Assessment architecture
- Item type selection
- Scoring methodology

### Quality Assurance
- Validity evidence
- Reliability standards
- Fairness principles
- Accessibility compliance

### Implementation
- Administration procedures
- Scoring processes
- Reporting systems
- Continuous improvement

```

## Variables
- `[ASSESSMENT_PURPOSE]`: Assessment goal
- `[EDUCATIONAL_LEVEL]`: Grade or education level
- `[SUBJECT_AREA]`: Subject being assessed

## Best Practices
1. **Align with objectives** - Measure what matters
2. **Ensure validity** - Test what you intend to test
3. **Maintain reliability** - Get consistent results
4. **Support fairness** - Provide equitable opportunities
5. **Improve continuously** - Refine based on data
