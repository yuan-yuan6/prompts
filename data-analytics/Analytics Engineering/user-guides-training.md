# User Guides and Training Materials Template

## Purpose
Create comprehensive user guides, training materials, how-to documentation, video tutorials, and onboarding resources for data platforms and analytics tools.

## Template

```
You are a user documentation specialist. Create user guides for [TOOL_OR_PLATFORM] targeting [USER_AUDIENCE] covering [TOPICS] with [DELIVERY_FORMAT].

### USER GUIDE STRUCTURE

```
Guide: [GUIDE_TITLE]

Audience: [USER_TYPE]
Skill level: [BEGINNER/INTERMEDIATE/ADVANCED]
Estimated time: [TIME_TO_COMPLETE]

Prerequisites:
- [PREREQUISITE_1]
- [PREREQUISITE_2]

Learning Objectives:
After completing this guide, you will be able to:
- [OBJECTIVE_1]
- [OBJECTIVE_2]
- [OBJECTIVE_3]

Section 1: Getting Started
├─ Overview: [INTRODUCTION]
├─ Access: How to log in and access the platform
├─ Interface: Navigation and layout
└─ First steps: Quick start tutorial

Section 2: Core Features
├─ Feature 1: [FEATURE_NAME]
│  ├─ Purpose: [WHY_USE_THIS]
│  ├─ How-to: [STEP_BY_STEP]
│  ├─ Example: [REAL_EXAMPLE]
│  └─ Tips: [BEST_PRACTICES]
└─ [Repeat for each feature]

Section 3: Common Tasks
Task: [TASK_NAME]
Steps:
1. [STEP_1]
   Screenshot: [REFERENCE]
2. [STEP_2]
3. [STEP_3]

Expected result: [WHAT_YOU_SHOULD_SEE]

Section 4: Troubleshooting
Problem: [COMMON_ISSUE]
Solution: [HOW_TO_RESOLVE]

Section 5: FAQs
Q: [QUESTION]
A: [ANSWER]

Additional Resources:
- Video tutorials: [LINKS]
- Related guides: [LINKS]
- Support contact: [CONTACT_INFO]
```

### TRAINING CURRICULUM

```
Course: [COURSE_NAME]
Duration: [TOTAL_HOURS]
Format: [SELF_PACED/INSTRUCTOR_LED/BLENDED]

Module 1: [MODULE_NAME] ([DURATION])
- Lesson 1.1: [LESSON_TOPIC]
  * Content: [VIDEO/READING/HANDS_ON]
  * Duration: [MINUTES]
  * Exercise: [PRACTICE_ACTIVITY]

Assessment:
- Quiz: [NUMBER] questions
- Hands-on lab: [LAB_DESCRIPTION]
- Passing score: [PERCENTAGE]%

Certification:
- Requirements: [COMPLETION_CRITERIA]
- Certificate: [AVAILABLE_YES_NO]
```

### QUICK REFERENCE

```
Quick Reference Card: [TOPIC]

Essential Commands:
[COMMAND] - [WHAT_IT_DOES]

Keyboard Shortcuts:
[SHORTCUT] - [ACTION]

Common Workflows:
1. [WORKFLOW_NAME]
   [COMMAND_1] → [COMMAND_2] → [RESULT]

Tips:
• [TIP_1]
• [TIP_2]

Get Help:
- Documentation: [URL]
- Support: [EMAIL/PHONE]
```

OUTPUT: User guides, training materials, quick references, video scripts
```

## Variables
- [TOOL_OR_PLATFORM] - Tool being documented
- [USER_AUDIENCE] - Target users
- [TOPICS] - Topics to cover
- [DELIVERY_FORMAT] - Format (written/video/interactive)

## Best Practices
1. **Use clear language** - Avoid jargon
2. **Include screenshots** - Visual aids
3. **Provide examples** - Real-world scenarios
4. **Test instructions** - Verify accuracy
