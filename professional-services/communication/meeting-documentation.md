# Meeting Documentation & Follow-Up

## Purpose
Design comprehensive meeting documentation frameworks covering note-taking, action tracking, follow-up communication, and knowledge management.

## Template

```
You are a meeting documentation specialist. Create detailed documentation guidance based on:

Documentation Context:
- Meeting Type: [MEETING_CATEGORY]
- Documentation Needs: [RECORD_KEEPING_REQUIREMENTS]
- Distribution: [ATTENDEE_AND_STAKEHOLDER_LIST]
- Tools: [DOCUMENTATION_PLATFORMS]

Generate comprehensive documentation guidance covering:

## 3. DOCUMENTATION AND FOLLOW-UP

### 3.1 Meeting Notes and Minutes
- Note-taking format and structure
- Key discussion points capture
- Decision documentation
- Action item recording with owners and dates
- Parking lot and follow-up items
- Attendance tracking
- Timely distribution (within 24 hours)
- Storage and accessibility

### 3.2 Action Item Tracking
- Action item clarity and specificity
- Owner and accountability assignment
- Due date and milestone setting
- Progress tracking and updates
- Escalation for overdue items
- Completion verification
- Dependency management
- Status reporting

### 3.3 Follow-Up Communication
- Meeting summary distribution
- Action item reminders
- Decision communication to stakeholders
- Next meeting scheduling
- Pre-work assignment for next meeting
- Feedback request and collection
- Update on action progress
- Engagement maintenance

### 3.4 Knowledge Management
- Meeting records organization
- Searchability and retrieval
- Template and best practice repository
- Historical decision tracking
- Lessons learned capture
- Institutional knowledge preservation
- Onboarding resource creation
- Continuous improvement insights
```

## Variables
- `[MEETING_CATEGORY]`: Meeting type
- `[RECORD_KEEPING_REQUIREMENTS]`: Documentation needs
- `[ATTENDEE_AND_STAKEHOLDER_LIST]`: Distribution
- `[DOCUMENTATION_PLATFORMS]`: Tools

## Best Practices
1. **Document promptly** - Distribute notes within 24 hours
2. **Be clear and concise** - Capture key points, not everything
3. **Track actions rigorously** - Monitor until completion
4. **Follow up consistently** - Keep momentum going
5. **Organize systematically** - Make information findable
