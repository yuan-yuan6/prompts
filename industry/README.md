## üöÄ Getting Started

### Choosing a Template

1. Browse the subcategories above to find relevant templates
2. Click on a template name to view its full content
3. Review the template structure, variables, and examples
4. Copy the template and fill in your specific variables

### Template Structure

Each template includes:

- **Purpose**: What the template is designed to accomplish
- **Template Body**: The full prompt with [VARIABLES] to customize
- **Variables**: Definitions and examples for each customizable field
- **Usage Examples**: Real-world scenarios showing the template in action
- **Customization Options**: Ways to adapt the template for different needs
