## Variables
- `[PROGRAM_TITLE]`: Program name
- `[EDUCATIONAL_LEVEL]`: Education level
- `[ACCREDITATION_STANDARDS]`: Accreditation requirements
- `[PROFESSIONAL_STANDARDS]`: Professional requirements
- `[ASSESSMENT_STRATEGY]`: Assessment approach
- `[EVALUATION_APPROACH]`: Evaluation method
- `[STANDARDS_FRAMEWORK]`: Standards system
- `[EXAM_FORMATS]`: Exam types
- `[MC_SPECIFICATIONS]`: Multiple choice specs
- `[PROJECT_REQUIREMENTS]`: Project criteria
- `[PORTFOLIO_CRITERIA]`: Portfolio standards
- `[EXEMPLARY_CRITERIA]`: Top performance level
- `[PROFICIENT_CRITERIA]`: Proficient level
- `[DEVELOPING_CRITERIA]`: Developing level
- `[BEGINNING_CRITERIA]`: Beginning level
- `[CONTENT_RUBRIC]`: Content assessment
- `[THINKING_RUBRIC]`: Critical thinking rubric
- `{NATIONAL_STANDARD_1-5}`: National standards
- `[COURSES_ADDRESSING_NS1]`: Courses for standard 1
- `{PROFESSIONAL_STANDARD_1-5}`: Professional standards
- `[PS1_COVERAGE]`: Standard 1 coverage
- `[PROGRAM_OUTCOME_1]`: Program outcome
- `{CONTRIBUTING_COURSE_1-4}`: Contributing courses
- `[ASSESSMENT_METHODS_PO1]`: Assessment methods

## Usage Example
Use for developing assessment frameworks, creating rubrics, mapping curriculum to standards, or ensuring alignment between outcomes and assessments.

## Customization Tips
- Align assessments to specific standards
- Use multiple assessment methods
- Develop clear, detailed rubrics
- Map outcomes across entire program
- Include both formative and summative assessments
- Ensure cultural responsiveness in assessment
