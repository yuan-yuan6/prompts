# Investment Performance Measurement Template

## Purpose
Measure and analyze investment performance through return calculation, benchmarking, attribution analysis, and risk-adjusted metrics to evaluate portfolio effectiveness and guide future decisions.

## Template

```
You are a performance measurement specialist with expertise in investment analysis and attribution. Create a detailed performance measurement framework based on the following information:

Portfolio Context:
- Company Name: [COMPANY_NAME]
- Investment Portfolio: [INVESTMENT_PORTFOLIO]
- Measurement Period: [EVALUATION_PERIOD]
- Benchmark: [PERFORMANCE_BENCHMARK]
- Cost of Capital: [WACC_RATE]
- Target Return: [HURDLE_RATE]

Investment Context:
- Strategic Objectives: [STRATEGIC_OBJECTIVES]
- Investment Categories: [INVESTMENT_TYPES]
- Risk Profile: [INVESTMENT_RISK_LEVEL]

Generate a comprehensive performance measurement framework that includes:

## PERFORMANCE MEASUREMENT AND ATTRIBUTION
### Portfolio Performance Evaluation
#### Return Calculation and Benchmarking
##### Time-Weighted and Money-Weighted Returns
###### Time-Weighted and Money-Weighted Returns
- Geometric and arithmetic return calculation
- Cash flow timing and impact analysis
- Benchmark selection and appropriateness
- Relative performance and active return
- Risk-adjusted return and information ratio
- Attribution analysis and source identification
- Performance persistence and consistency
- Statistical significance and confidence

###### Value Creation and Destruction Analysis
- Economic value added (EVA) and ROIC analysis
- Market value added (MVA) and TSR measurement
- Value creation and destruction attribution
- Business unit and segment contribution
- Strategic initiative and project impact
- Capital allocation efficiency and effectiveness
- Opportunity cost and alternative assessment
- Stakeholder value and return analysis

##### Risk-Adjusted Performance Metrics
###### Portfolio Risk Assessment

- Maximum drawdown and downside risk
- Tracking error and active risk
- Risk contribution and decomposition
- Stress testing and scenario analysis
- Risk capacity and appetite alignment

###### Risk-Return Efficiency
- Sharpe ratio and risk-adjusted return
- Information ratio and active efficiency
- Treynor ratio and systematic risk adjustment
- Jensen's alpha and abnormal return
- Sortino ratio and downside risk focus
- Calmar ratio and maximum drawdown
- Risk-return scatter and efficient frontier
- Peer comparison and relative performance

#### Performance Attribution and Analysis
##### Asset Allocation and Security Selection
- Strategic asset allocation contribution
- Tactical allocation and timing impact
- Security selection and stock picking
- Interaction effect and cross-terms
- Currency and foreign exchange impact
- Sector and industry allocation
- Style and factor exposure
- Active management and skill assessment

##### Factor-Based Attribution
- Risk factor and style exposure
- Market factor and systematic return
- Size, value, and growth factors
- Quality, momentum, and volatility factors
- Sector and industry factors
- Country and regional factors
- Currency and commodity factors
- Custom factor and specific risks
```

## Variables
- `[COMPANY_NAME]`: Organization name
- `[INVESTMENT_PORTFOLIO]`: Portfolio description
- `[EVALUATION_PERIOD]`: Measurement period
- `[PERFORMANCE_BENCHMARK]`: Benchmark reference
- `[WACC_RATE]`: Cost of capital
- `[HURDLE_RATE]`: Target return
- `[STRATEGIC_OBJECTIVES]`: Strategic goals
- `[INVESTMENT_TYPES]`: Investment categories
- `[INVESTMENT_RISK_LEVEL]`: Risk profile

## Usage Example
Use for investment performance evaluation, portfolio review, attribution analysis, benchmark comparison, or performance reporting.

## Customization Tips
- Adapt return metrics to investment type and horizon
- Include industry-specific performance benchmarks
- Add strategic objective achievement metrics
- Incorporate ESG and sustainability performance measures
- Include qualitative and non-financial value creation
- Add peer comparison for competitive context
