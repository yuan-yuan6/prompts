# Compliance Monitoring & Testing Template Generator

## Purpose
Create comprehensive compliance monitoring and surveillance systems covering continuous monitoring, performance measurement, audit processes, and effectiveness validation.

## Template

```
You are a compliance monitoring and testing specialist. Create comprehensive monitoring strategy based on:

Context:
- Organization: [COMPANY_NAME]
- Monitoring Scope: [MONITORING_SCOPE]
- Risk Areas: [KEY_RISK_AREAS]

## COMPLIANCE MONITORING AND SURVEILLANCE
### Continuous Monitoring Framework
- Real-time monitoring system
- Automated detection and alerting
- Key performance indicator tracking
- Exception and anomaly detection
- Behavioral analysis and trending
- Predictive analytics integration
- Dashboard and visualization
- Mobile and remote monitoring

### Performance Measurement
- Compliance effectiveness metrics
- Leading and lagging indicators
- Management reporting dashboards
- Regulatory reporting requirements

### Audit and Assessment Process
- Internal audit planning and execution
- Quality assurance validation
- External assessment coordination
- Independent certification pursuit

### Investigation and Analysis
- Incident detection and triage
- Root cause analysis techniques
- Evidence management
- Documentation and reporting
```

## Variables
- `[COMPANY_NAME]`: Organization name
- `[MONITORING_SCOPE]`: Areas under monitoring
- `[KEY_RISK_AREAS]`: Critical compliance risks

## Best Practices
1. **Automate monitoring** - Use technology for continuous surveillance
2. **Focus on leading indicators** - Predict issues before they occur
3. **Test controls regularly** - Validate effectiveness of compliance controls
4. **Document findings** - Maintain comprehensive audit trails
5. **Act quickly** - Remediate issues promptly
