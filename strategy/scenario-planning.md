# Strategic Scenario Planning Framework

## Purpose
Develop multiple plausible future scenarios and corresponding strategic responses to navigate uncertainty, build resilience, and position the organization to thrive across different possible futures rather than betting on a single forecast.

## Quick Start

**Want to prepare for uncertain futures quickly?** Here's the minimal approach:

### When to Use This
- Facing high uncertainty about future market conditions
- Long-term strategic decisions with irreversible commitments
- Disruptive changes threatening current business model
- Need to align stakeholders on strategic direction
- Planning major investments or transformations

### Minimal Example
```
Company: Regional airline
Planning Horizon: 5 years
Key Uncertainties: Fuel prices, demand recovery post-pandemic

Quick Scenarios (2x2 Matrix):
Axis 1: Fuel Prices (High/Low)
Axis 2: Travel Demand (Strong Recovery/Weak Recovery)

Scenario 1: "Blue Sky" (High demand, Low fuel) - Aggressive expansion
Scenario 2: "Turbulence" (High demand, High fuel) - Premium positioning
Scenario 3: "Headwinds" (Low demand, High fuel) - Defensive consolidation
Scenario 4: "Grounded" (Low demand, Low fuel) - Wait and see

Adaptive Strategy:
- Build flexible capacity (leases not purchases)
- Develop premium service capability
- Maintain cost efficiency programs
- Create partnership options

Result: Survived pandemic better than competitors using adaptive approach
```

### Basic 5-Step Workflow
1. **Identify Key Uncertainties** - What's most uncertain and impactful
2. **Develop Scenario Frameworks** - Create 3-4 distinct, plausible futures
3. **Build Detailed Scenarios** - Flesh out each future world
4. **Derive Strategic Implications** - What succeeds/fails in each scenario
5. **Develop Adaptive Strategy** - Strategy that works across scenarios

**Time to complete**: 2-3 days for basic scenarios, 3-4 weeks for comprehensive planning

**Pro tip**: Focus on what could happen, not what you want to happen - challenge assumptions

## Best Practices

### Do's ✅
- **Make Scenarios Plausible**: Must be believable, not sci-fi
- **Make Scenarios Different**: Scenarios should diverge meaningfully
- **Give Scenarios Names**: Memorable names help internalize scenarios
- **Involve Diverse Perspectives**: Include contrarian voices
- **Use Scenarios, Don't Shelve**: Reference in strategic decisions
- **Update Regularly**: Scenarios become stale; refresh annually
- **Focus on Critical Uncertainties**: Don't boil the ocean

### Don'ts ❌
- **Don't Predict the Future**: Scenarios are not forecasts
- **Don't Create Too Many**: 3-4 scenarios is optimal; more causes paralysis
- **Don't Make Scenarios Probability-Weighted**: Avoid "most likely" thinking
- **Don't Ignore Uncomfortable Scenarios**: Challenge is the point
- **Don't Let Politics Drive**: Scenarios should challenge assumptions
- **Don't Create Linear Extrapolations**: Include discontinuities

## Common Scenario Planning Methods

### 1. **Intuitive Logics (Shell Method)**
- Identify focal question
- Find critical uncertainties
- Build 3-4 narrative scenarios
- Derive strategic implications

### 2. **Probable, Possible, Plausible, Preferable (4Ps)**
- Probable: Baseline forecast
- Possible: What could happen
- Plausible: What's believable
- Preferable: What we want

### 3. **Scenario Axes (2x2 Matrix)**
- Select two critical uncertainties
- Create four scenarios at extremes
- Explore implications

### 4. **Morphological Analysis**
- Identify key variables
- Define possible states for each
- Combine into scenarios
