# Coalition Building & Collective Impact

## Purpose
Framework for building multi-stakeholder coalitions, collective impact initiatives, shared measurement systems, mutually reinforcing activities, continuous communication, and backbone support organizations.

## Quick Start

### Minimal Example
```
Build coalition of 25 organizations addressing childhood poverty including shared vision, common agenda, shared measurement system, coordinated activities, continuous communication structure, and backbone organization ensuring collective impact.
```

### Basic 3-Step Workflow
1. **Coalition Formation** - Vision, membership, structure (2-3 months)
2. **Infrastructure Building** - Governance, measurement, communication (3-6 months)
3. **Collaborative Action** - Coordinated activities, learning, adaptation (multi-year)

---

## Key Components

- Common agenda and shared vision
- Shared measurement systems
- Mutually reinforcing activities
- Continuous communication
- Backbone support organization
- Governance structure
- Resource mobilization
- Conflict resolution
- Sustainability planning
