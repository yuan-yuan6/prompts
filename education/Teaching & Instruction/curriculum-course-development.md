## Variables
- `[PROGRAM_TITLE]`: Program name
- `[EDUCATIONAL_LEVEL]`: Education level
- `[PROGRAM_DURATION]`: Program length
- `[DELIVERY_FORMAT]`: Delivery mode
- `[CREDIT_HOURS]`: Total credits
- `[COURSE_TITLE]`: Course name
- `[COURSE_NUMBER]`: Course ID
- `[COURSE_CREDITS]`: Course credits
- `[COURSE_PREREQUISITES]`: Prerequisites
- `[DELIVERY_MODE]`: How delivered
- `[PEDAGOGICAL_APPROACH]`: Teaching method
- `[TECHNOLOGY_INTEGRATION]`: Tech use
- `[ASSESSMENT_STRATEGY]`: Assessment approach
- `{YEAR1_SEM1_COURSE1-5}`: Course names
- `{CREDITS_1-10}`: Credit hours
- `[FOUNDATION_COURSES]`: Foundation courses
- `[CORE_COURSES]`: Core courses
- `[COURSE_DESCRIPTION]`: Course overview
- `{COURSE_OUTCOME_1-5}`: Course outcomes
- `[MODULE_1_TITLE]`: Module title
- `{MODULE_1_TOPIC_1-4}`: Module topics
- `{MODULE_1_ACTIVITY_1-4}`: Learning activities
- `[INTERACTIVE_ELEMENTS]`: Interactive components
- `[LMS_PLATFORM]`: LMS system

Use for developing detailed course designs, creating module plans, designing learning activities, or integrating technology into courses.

- Align modules to learning outcomes
- Balance theoretical and practical learning
- Include diverse learning activities
- Integrate technology purposefully
- Build in formative assessment opportunities
- Design for different learning styles
