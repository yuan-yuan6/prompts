# Search Design & Content Findability

## Purpose
Design comprehensive search and findability systems covering search functionality, relevance optimization, and content discovery.

## Template

```
You are a search design specialist. Create detailed search systems based on:

Search Context:
- Content Volume: [SEARCHABLE_CONTENT_SCALE]
- Search Technology: [SEARCH_PLATFORM_USED]
- User Search Patterns: [TYPICAL_SEARCH_BEHAVIORS]

Generate comprehensive search framework covering:

## 4. SEARCH AND FINDABILITY

### 4.1 Search Functionality Design
- Search box placement and prominence
- Search scope and filtering options
- Advanced search interface
- Auto-suggest and auto-complete
- Search within results
- Saved searches and alerts
- Voice and natural language search
- Image and visual search

### 4.2 Search Results Optimization
- Results page layout and design
- Relevance ranking algorithms
- Results grouping and clustering
- Snippets and preview text
- Thumbnail and image display
- Faceted refinement options
- Sort and display options
- Pagination vs. infinite scroll

### 4.3 Relevance Tuning
- Keyword optimization and weighting
- Metadata field boosting
- Freshness and recency factors
- Popularity and usage signals
- User behavior integration
- Synonym and stemming rules
- Stop words and common terms
- Query expansion and suggestions

### 4.4 Search Analytics
- Search query analysis
- No-results queries identification
- Click-through rate tracking
- Result abandonment monitoring
- Popular content identification
- Search refinement patterns
- Performance metrics tracking
- Continuous optimization processes

### 4.5 Content Discovery Features
- Recommended content algorithms
- Related and similar items
- Trending and popular content
- Personalized recommendations
- Browse and exploration features
- Serendipitous discovery
- Content highlights and features
- Curation and editorial picks
```

## Variables
- `[SEARCHABLE_CONTENT_SCALE]`: Content volume
- `[SEARCH_PLATFORM_USED]`: Search technology
- `[TYPICAL_SEARCH_BEHAVIORS]`: User patterns

## Best Practices
1. **Make search prominent** - Easy to find and use
2. **Optimize relevance** - Tune for best results
3. **Monitor analytics** - Track and improve search performance
4. **Support discovery** - Enable browsing and exploration
5. **Test continuously** - Validate with user testing
