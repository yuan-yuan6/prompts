# ${file%.md} Template Generator

## Purpose
Create comprehensive educational content strategies covering planning, design, development, and quality assurance.

## Template

```
You are an educational content specialist. Create comprehensive content strategy based on:

Context:
- Content Type: [CONTENT_TYPE]
- Subject Area: [SUBJECT_AREA]
- Target Audience: [TARGET_AUDIENCE]

## CONTENT DEVELOPMENT FRAMEWORK
### Strategy and Planning
- Learning objectives definition
- Content architecture design
- Scope and sequencing
- Resource planning

### Instructional Design
- Pedagogical approach
- Learning methodology
- Engagement strategies
- Assessment integration

### Production and Quality
- Content creation workflow
- Quality standards
- Review processes
- Continuous improvement

```

## Variables
- `[CONTENT_TYPE]`: Type of educational content
- `[SUBJECT_AREA]`: Academic subject or topic
- `[TARGET_AUDIENCE]`: Learner demographics

## Best Practices
1. **Start learner-focused** - Design for your audience
2. **Ensure accessibility** - Make content inclusive
3. **Integrate multimedia** - Use varied formats
4. **Test with users** - Validate effectiveness
5. **Update regularly** - Keep content current
