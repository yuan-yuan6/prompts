## 1. PROFESSIONAL FORMATTING

### Industry Standard Format
#### Basic Requirements
- Font: Courier or Courier New, 12-point
- Margins: 1.5" left, 1" right, top, bottom
- Page numbers: Top right, 0.5" from top
- Scene headings: All caps, flush left
- Character names: All caps, centered above dialogue
- Dialogue: Centered, 2.5" from left margin
- Parentheticals: Centered in parentheses
- Action: Flush left, single-spaced paragraphs

### Scene Heading Format
#### Slug Line Construction
- INT. or EXT. designation
- Location specification
- Time of day (DAY/NIGHT)
- Continuous action notation (CONTINUOUS)
- Specific time notation when needed
- Moving vehicle notation (MOVING)
- Sub-location indication
- Clarity and brevity balance

### Character and Dialogue
#### Dialogue Formatting
- Character name all caps and centered
- Dialogue centered and double-spaced from action
- Parenthetical usage and placement
- (CONT'D) when dialogue continues
- (V.O.) for voice-over
- (O.S.) for off-screen
- (O.C.) for off-camera
- (PRE-LAP) for dialogue before scene

### Action Line Format
#### Description Formatting
- Present tense and active voice
- Single-spaced paragraphs
- Double-space between paragraphs
- Character name capitalization on introduction
- Sound effect capitalization
- Important action or prop capitalization
- White space for readability
- Paragraph length variation

## 2. TITLE PAGE AND FRONT MATTER

### Title Page Format
#### Standard Elements
- Title: Centered, all caps, 1/3 down page
- "Written by" centered below title
- Author name centered below
- Contact information: Lower left corner
- WGA registration or copyright: Lower right
- Draft date: Lower right
- Based on source material if applicable
- Contact: Agent/manager if applicable

### Additional Pages
#### Optional Front Matter
- Logline page
- Synopsis page
- Character list
- Treatment document
- Pitch deck
- Previous draft notes
- Series bible for TV
- Episode guide for series

## 3. SUBMISSION STANDARDS

### Page Count Requirements
#### Length Guidelines
- Feature film: 90-120 pages
- Drama features: 100-120 pages
- Comedy features: 90-110 pages
- Action features: 100-120 pages
- TV hour drama: 50-60 pages
- TV half-hour comedy: 25-35 pages
- Pilot episode: Genre-appropriate length
- Spec vs. production draft differences

### Professional Presentation
#### Submission Package
- Clean, error-free text
- Proper formatting throughout
- No typos or grammatical errors
- Professional binding (brads, not bound)
- Cover page information accuracy
- Page count appropriateness
- Industry-standard software output
- PDF for digital submission
