# Query Optimization - Overview & Navigation

## Purpose
This is a navigation hub for the query optimization template collection. The comprehensive query optimization template has been split into focused sub-prompts for easier use and better clarity.

## Quick Start

**Choose Your Template:**

Browse the specialized templates below and select the one that matches your specific needs. Each template is focused, easy to use, and can be applied independently.

### 1. Query Analysis & Profiling Template
**File:** `query-analysis-profiling.md`

**Purpose:** Analyze query performance using execution plans, profiling tools, and performance metrics to identify optimization opportunities.

**Best for:** Profiling, Execution Plan, Analysis

---

### 2. Query Optimization Strategies Template
**File:** `query-optimization-strategies.md`

**Purpose:** Apply query optimization techniques including query rewriting, join optimization, subquery elimination, and performance tuning.

**Best for:** Optimization, Query Rewriting, Join

---

### 3. Database Indexing Strategies Template
**File:** `database-indexing-strategies.md`

**Purpose:** Design and implement indexing strategies using B-tree, columnstore, covering indexes, and partitioning for query performance.

**Best for:** Index, Indexing, B-tree

---

### 4. Query Performance Monitoring Template
**File:** `query-performance-monitoring.md`

**Purpose:** Implement query performance monitoring, automated statistics updates, and continuous optimization for database systems.

**Best for:** Monitoring, Performance, Statistics

---
