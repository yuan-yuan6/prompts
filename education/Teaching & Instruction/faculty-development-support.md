# Faculty Development & Support Template

## Purpose
Create comprehensive faculty development programs and resource plans including qualifications, professional development, technology infrastructure, and support systems for educational programs.

## Template

```
You are an expert in faculty development and educational resource planning with expertise in professional development and organizational capacity building. Create a detailed faculty development and resource plan based on:

Program Context:
- Program Title: [PROGRAM_TITLE]
- Institution Type: [INSTITUTION_TYPE]
- Program Size: [PROGRAM_SIZE]
- Faculty Count: [FACULTY_COUNT]

Development Context:
- Professional Development Focus: [PROF_DEV_FOCUS]
- Resource Requirements: [RESOURCE_NEEDS]
- Technology Integration: [TECHNOLOGY_INTEGRATION]
- Support Services: [SUPPORT_SERVICES]

Generate a comprehensive faculty development and resource plan:
```

## Variables
- `[PROGRAM_TITLE]`: Program name
- `[INSTITUTION_TYPE]`: Type of institution
- `[PROGRAM_SIZE]`: Program scale
- `[FACULTY_COUNT]`: Number of faculty
- `[PROF_DEV_FOCUS]`: Development priorities
- `[RESOURCE_NEEDS]`: Resource requirements
- `[TECHNOLOGY_INTEGRATION]`: Tech needs
- `[SUPPORT_SERVICES]`: Support systems
- `[EDUCATION_REQUIREMENTS]`: Required education
- `[EXPERIENCE_REQUIREMENTS]`: Required experience
- `[TEACHING_CREDENTIALS]`: Teaching certifications
- `[RESEARCH_REQUIREMENTS]`: Research expectations
- `[TECHNOLOGY_SKILLS]`: Tech competencies
- `[FULLTIME_FACULTY_COUNT]`: Full-time positions
- `[PARTTIME_FACULTY_COUNT]`: Part-time positions
- `[GTA_COUNT]`: Teaching assistants
- `[ADVISOR_COUNT]`: Advisors needed
- `[CLASSROOM_COUNT]`: Number of classrooms
- `[LAB_COUNT]`: Laboratory spaces
- `[LMS_REQUIREMENTS]`: LMS needs
- `[BOOK_COLLECTION_SIZE]`: Library collection size

## Usage Example
Use for faculty recruitment planning, professional development design, resource allocation, or infrastructure planning for educational programs.

## Customization Tips
- Tailor qualifications to discipline needs
- Include mentorship and peer support
- Plan for ongoing technology training
- Budget for professional development
- Ensure adequate support staff
- Provide diverse development opportunities
