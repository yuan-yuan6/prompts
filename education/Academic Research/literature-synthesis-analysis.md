# Literature Synthesis and Analysis

## Purpose
Create comprehensive, rigorous analysis and synthesis frameworks for integrating research evidence across multiple studies. This template guides quantitative meta-analysis, qualitative thematic synthesis, mixed-methods integration, evidence grading, and pattern identification to generate meaningful insights and evidence-based conclusions.

## Best Practices

1. **Pre-specify analysis plan** - Document approach before beginning
2. **Use appropriate methods** - Match synthesis to data type and question
3. **Assess and explain heterogeneity** - Don't ignore variation
4. **Conduct sensitivity analyses** - Test robustness of findings
5. **Grade evidence quality** - Apply GRADE or similar framework
6. **Maintain transparency** - Document all decisions and methods
7. **Seek statistical expertise** - Consult methodologists for complex analyses
8. **Use validated tools** - Apply established frameworks and software
9. **Consider multiple perspectives** - Integrate diverse evidence types
10. **Identify gaps clearly** - Specify future research needs

## Tips for Success

- Begin with clear analytic questions
- Organize data systematically before analysis
- Use established synthesis frameworks (GRADE, CERQual)
- Calculate inter-rater reliability for coding (κ ≥ 0.7)
- Report heterogeneity even when low
- Create visual displays (forest plots, theme diagrams)
- Distinguish statistical from clinical significance
- Acknowledge limitations transparently
- Consider both quantitative and qualitative evidence
- Allow 12-16 weeks for comprehensive synthesis and analysis
