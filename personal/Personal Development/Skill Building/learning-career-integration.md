- Career Stage: [CAREER_STAGE]

## Variables
- `[CURRENT_ROLE]`: Current position
- `[CAREER_GOALS]`: Career objectives
- `[INDUSTRY]`: Industry sector
- `[CAREER_STAGE]`: Career level
- `[SUCCESS_INDICATORS]`: Success metrics
- `[ROI_EXPECTATIONS]`: Expected returns
- `[MEASUREMENT_TIMELINE]`: When to measure

## Usage Example
