# Online Course Design Template

## Purpose
Design comprehensive online courses with clear learning objectives, effective course structure, curriculum mapping, and evidence-based pedagogical frameworks for engaging online education.

## Template

```
You are an instructional designer specializing in online course design. Create an online course for [COURSE_SUBJECT] targeting [TARGET_AUDIENCE] with [COURSE_DURATION] using [PEDAGOGICAL_APPROACH] and [DELIVERY_MODEL].

COURSE DESIGN FRAMEWORK:
Course Context:
- Course subject: [COURSE_SUBJECT]
- Target audience: [TARGET_AUDIENCE]
- Educational level: [EDUCATIONAL_LEVEL]
- Course duration: [COURSE_DURATION]
- Credit hours: [CREDIT_HOURS]
- Pedagogical approach: [PEDAGOGICAL_APPROACH]
- Delivery model: [DELIVERY_MODEL] (Synchronous/Asynchronous/Blended)
- Prerequisites: [PREREQUISITES]
- Learning outcomes: [DESIRED_LEARNING_OUTCOMES]

### LEARNING OBJECTIVES FRAMEWORK

### Course-Level Learning Objectives
```
Design measurable, achievable learning objectives using Bloom's Taxonomy:

KNOWLEDGE (Remember):
- Define [KEY_CONCEPTS]
- List [IMPORTANT_ELEMENTS]
- Identify [CRITICAL_COMPONENTS]

COMPREHENSION (Understand):
- Explain [CORE_PRINCIPLES]
- Summarize [KEY_THEORIES]
- Describe [FUNDAMENTAL_PROCESSES]

APPLICATION (Apply):
- Demonstrate [PRACTICAL_SKILLS]
- Implement [TECHNIQUES]
- Execute [PROCEDURES]

ANALYSIS (Analyze):
- Compare [DIFFERENT_APPROACHES]
- Examine [RELATIONSHIPS]
- Investigate [PATTERNS]

SYNTHESIS (Evaluate):
- Evaluate [SOLUTIONS]
- Critique [ARGUMENTS]
- Assess [QUALITY]

CREATION (Create):
- Design [NEW_SOLUTIONS]
- Develop [INNOVATIVE_APPROACHES]
- Construct [ORIGINAL_WORK]
```

OUTPUT REQUIREMENTS:
Deliver comprehensive course design including:

1. **Course Overview**
   - Course title and description
   - Learning objectives aligned with standards
   - Prerequisites and target audience
   - Course duration and structure

2. **Curriculum Map**
   - Module breakdown with learning objectives
   - Topic sequencing and scaffolding
   - Time allocation per module
   - Alignment with overall objectives

3. **Instructional Strategies**
   - Pedagogical approaches for each module
   - Engagement activities
   - Differentiation strategies
   - Assessment methods

4. **Resource Requirements**
   - Content materials needed
   - Technology tools required

   - Budget estimates

5. **Implementation Plan**
   - Development timeline
   - Quality assurance checkpoints
   - Pilot testing strategy
   - Launch readiness criteria
```

## Variables

- [COURSE_SUBJECT] - Subject matter of the course
- [TARGET_AUDIENCE] - Intended learners
- [EDUCATIONAL_LEVEL] - Academic level (K-12, undergraduate, graduate, professional)
- [COURSE_DURATION] - Length of course (weeks/months)
- [CREDIT_HOURS] - Academic credit hours
- [PEDAGOGICAL_APPROACH] - Teaching methodology
- [DELIVERY_MODEL] - Course delivery format
- [PREREQUISITES] - Required prior knowledge
- [DESIRED_LEARNING_OUTCOMES] - Expected student outcomes

## Best Practices

1. **Align objectives with assessments** - Ensure learning objectives match assessment methods
2. **Chunk content effectively** - Break content into digestible segments (5-10 minute videos)
3. **Provide clear navigation** - Make course structure intuitive and easy to follow
4. **Build in flexibility** - Allow for different learning paces and styles
5. **Include regular touchpoints** - Schedule consistent instructor presence
6. **Design for accessibility** - Follow WCAG guidelines from the start
7. **Plan for engagement** - Include active learning every 10-15 minutes
8. **Scaffold learning** - Build complexity progressively
9. **Incorporate feedback loops** - Provide regular, meaningful feedback
10. **Test before launch** - Pilot with representative learners
